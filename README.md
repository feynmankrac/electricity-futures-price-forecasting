## Overview
Forecasting electricity futures prices in France and Germany using multi-source datasets.

## Objective
- Explain daily variations in electricity futures prices
- Integrate meteorological, energy and trade-related variables
- Evaluate predictive power using rank-based metrics

## Methodology
- Tabular regression models
- Feature engineering on heterogeneous data sources
- Evaluation using Spearman correlation
